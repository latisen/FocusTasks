# FocusTasks
En grund till ett kraftfullt Obsidian‑plugin för uppgiftshantering inspirerat av OmniFocus.

## Funktioner (grund)
- Skannar alla anteckningar efter uppgifter som `- [ ] Uppgiftsnamn` eller `- [x] Klart`.
- Sidopanel som listar uppgifter.
- Enkel filterknapp för att visa/dölja klara uppgifter.

## Kom igång
1. Installera beroenden:
	- `npm install`
2. Bygg pluginet:
	- `npm run build`
3. Kopiera `main.js`, `manifest.json` och `styles.css` till din vault:
	- `.obsidian/plugins/focus-tasks/`

## Utveckling
- Starta watch‑läge:
  - `npm run dev`

## Nästa steg (förslag)
- Projekt/perspektiv
- Inbox
- Kontext/tags
- Skjuta upp/planerade datum
- Snabb öppning till rad i fil
